File Navigator

Click here to start demo

This example show a simple file navigator.

The code is simplified for educational purpouses but can be used as-is in your applications.
It is made out of five files grouped into a framed page.

  • navigator.msp: the frame page
  • nav_top.msp: frame 0 - banner and form for choosing start folder (root)
  • nav_left.msp: frame 1 - list of folders
  • nav_body.msp: frame 2 - list of files in current folder
  • nav_edit.msp: external text editor
