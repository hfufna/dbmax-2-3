This demo application illustrates 3 different ways of programming in MaxScript: CGI-mode, MaxObjects and Ajax

CGI-Mode
Everything is done in a single page.
The page is reloaded every time the display is updated by the timer (written in javascript).

MaxObject
Still a single page, but that calls itself recurively. Sane as above, t page is reloaded every time the display is updated.

Ajax-mode
Asynchronous mode: the main page is never reloaded. All actions are performed form an other page (the actuator) that is permanently cached.
