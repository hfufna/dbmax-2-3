Maintenance page

This page is intended for the notice board administrator.

In a real application should be accessible only to authorized users or from selected IP addresses.
Under Xitami this can be easily done by using defaults.aut (authorization file).

In effects a data-based login system, with a table of users and roles, would be a better solution.
To keep this example as simple as possible this part of the application has been omitted.

From this page you can perform these tasks:

  • Add a new record (a notice)
     
  • List and edit all records
