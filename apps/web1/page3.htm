Targeting mobile devices

Developing an application for a mobile device is not simple: at least if we mean an application that actually runs and resides on the devices itself.

Today almost every smartphone has an internet connection: so why not to develope something that will run on the device but inside its browser?

Discover how easy is to build a web application for mobile devices with Antartis, our application framework.
